# geekbrains-site. Vue-CLI-Route
### 1. Установить модули
```
npm install
```
### 2. Запустить сервер в терминале 1 
```
cd server
nodemon server.js
```
###3. Запустить клиент в терминале 2
```
npm run serve
```
### Компиляция и минификация для продакшена
```
npm run build
```
### Lints and fixes files
```
npm run lint
```
### Запуск графического интерфейса Vue-CLI
```
vue ui
```
